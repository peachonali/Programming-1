# Course-Project-PROGRAMMINH-1 โปรเจครายวิชา PROGRAMING language C
ชื่อโปรเจค ตู้กดน้ำอเนประสงค์  

มีความสนใจว่า หลังจากที่ได้เรียนภาษา C และ อยากทราบว่าถ้า เราใช้ภาษานี้ในการเขียนจำลองการทำงานของ ตู้กดน้ำ ที่แสดงผ่าน ทางหน้าจอ จะเป็นยังไง ซึ่งอาจจะเป็นโปรเจคขนาดเล็ก โค้ดไม่มาก แต่เป็นจุดเริ่มต้นของผม

เนื้อหา ประกอบด้วย ตู้กดน้ำที่จะกำหนดให้ 1 ใส่ขวดน้ำที่มีอยู่ในตู้ 2 ยอดเงินที่มีอยู่ในตู้  3 เติม
น้ำใส่ตามจำนวนขวด 4 ใส่เหรียญเข้าตู้ 5 การแจ้งเตือนว่าไม่สามารถเติมน้ำได้เนื่องจากเงินไม่พอ 6 
สถานะของขวดน้ำที่เหลืออยู่ 7 ยอดเงินที่เหลือในตู้ 8 ระบบแจ้งเตือนผู้ดูแลว่ามีการเติมน้ำ 9 การ
บันทึกประวัติการทำรายการ 10 คำนวณเงินทอนเมื่อจ่ายเงินเกินจำนวนขวด 

ปัญหาที่ต้องการแก้ไข คือ การกดน้ำเปล่าที่ตู้ไม่สามารถที่จะ บอกจำนวนเงินที่สามารถเติมได้อีกกี่ขวด หรือ ไม่มีการแจ้งเตือนทาง เจ้าของบริษัทว่ามีการกดน้ำดื่มอยู่ 

7 ขอบเขต Product 
7.1 สามารถแจ้งเตือนสถานะการอยู่ระหว่างกดได้ 
7.2 สถานะขวดน้ำที่เหลืออยู่ 
7.3 ระบบแจ้งเตือนผู้ดูแลว่ามีคนกดน้ำ 
7.4 การบันทึกประวัติการทำรายการ 
7.5 คำนวณเงินทอนหลังจากกดน้ำไม่หมดตามที่ใส่ขวด 

8 หลักการทำงาน  
เริ่มต้นจะมีโหมดให้เลือกว่า จะเติมน้ำ หรือ เติมเงิน หรือ กดน้ำ หรือออกจากโปรแกรม จากนั้น 
ให้เลือกตามโหมดที่ต้องการ แล้ว เข้าสู่การทำงานของนั่นๆ แล้วทำการทำรายการต่อ เพื่อจะ 
เลือกใช้ตามโหมดที่เราต้องการ
